# Dotfiles

## What it contains

The dotfiles contain all of my preferred applications that I need to install on every freshly installed MacOS and Omarchy system.

Clone the repo and install stow firstly, then you can start:

```bash
git clone https://github.com/TheRaigeki/dotfiles.git
cd dotfiles
```

## MacOS

Use the README_MACOS.md file to install and configure everything on a newly installed machine.
